# reactChat

A little chat app with node.js  and angular

## use technologies

- npm
- nodejs
- react 19
- socket.io
- ngX Starter Kit https://github.com/ngx-rocket/starter-kit

## Installation
```bash
// API start
cd .\API\
npm install
node .\src\index.js
npm install
// UI start
cd .\UI\
npm install
npm run dev
```
